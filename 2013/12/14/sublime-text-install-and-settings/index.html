安装插件

  • 按下Shift + Command + P (Mac)或者Shift + Ctrl + P(Windows)调出命令面板
  • 输入install，你会见到Package Control: Install Package，选中它按下Enter
  • 等待片刻，会弹出一个插件列表，输入需要安装的插件名字（如：Emmet），按下Enter安装它
  • 这样就完成了，就这么简单！
